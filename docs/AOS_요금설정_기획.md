# AOS 요금설정 기획

## 목적
상품등록 후 실제 예약금액 계산 기준을 관리하는 화면입니다.

## 구성
- 기본요금
- 기간별 요금
- 개별일자 요금
- 옵션요금
- 탑승지별 요금

## 기간별 요금
평일, 주말, 성수기, 극성수기처럼 기간과 요일 기준으로 요금을 일괄 적용합니다.

입력항목:
- 우선순위
- 적용구분
- 적용기간
- 요일
- 대인요금
- 소인요금
- 예약금
- 최소인원
- 최대인원
- 사용여부
- 메모

## 우선순위
1. 개별일자 요금
2. 극성수기 요금
3. 성수기 요금
4. 주말요금
5. 평일요금
6. 기본요금

## 옵션 적용방식
- 예약건당 1회 적용
- 예약인원 전체 자동적용
- 고객 수량선택

## 예약 후 요금변경
예약 발생 후 요금을 변경해도 기존 예약금액에는 자동 반영하지 않습니다.
신규 예약부터 변경된 요금이 적용됩니다.
