# AOS 전체기획 정리

## 목적
AOS는 여행사가 상품등록, 일정관리, 요금설정, 예약접수현황, 판매사 연동, 정산을 한 흐름으로 관리하는 여행사 SaaS 관리자 플랫폼입니다.

## 핵심 흐름
```text
상품등록 → 일정관리 → 요금설정 → 홈페이지 노출 → 예약접수 → 결제/입금 확인 → 알림톡 발송 → 출발자명단 → 출발완료 → 정산
```

## 주요 메뉴
1. 상품관리: 상품등록, 상품리스트, 일정표등록, 요금설정
2. 예약관리: 예약접수현황, 예약상세, 결제/입금관리, 취소/환불관리
3. 판매사 관리: 판매사 등록, 상품공급 설정, 커미션 설정
4. 알림톡/문자관리: 예약접수, 예약확정, 결제요청, 출발안내
5. 정산관리: 상품별, 판매사별, 공급사별, 지자체 정산

## UI 원칙
- 표 중심의 운영형 관리자 UI
- 검색/필터는 상단 배치
- 상태값은 컬러 뱃지로 표시
- 리스트 우측에 상세/관리/알림/복사/삭제 등 기능 버튼 배치
