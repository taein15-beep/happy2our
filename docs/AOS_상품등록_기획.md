# AOS 상품등록 기획

## 목적
상품등록 화면은 여행상품의 기본정보를 등록하는 화면입니다. 일정관리와 요금관리는 상품등록 후 별도 페이지에서 입력합니다.

## 주요 항목
- 대분류/중분류/소분류 카테고리
- 추가노출 카테고리도 대분류/중분류/소분류 적용
- 상품명, 상품안내, 대표요금, 판매기간
- 상품코드, 담당자, 여행지역
- 예약최소인원, 최소출발인원, 예약마감인원, 예약마감일
- 상품라벨: 태그 입력 방식
- 상품이미지: 다중 등록, 대표 설정, 순서 변경
- 포함사항, 불포함사항, 상세안내, 이용안내, 취소/환불규정
- 판매사 공급/커미션 설정
- 관리자메모, 판매사 전달메모, 관리자용 파일첨부

## 정책
- 상품등록은 빠른 등록을 목표로 합니다.
- 실제 판매요금은 요금설정에서 관리합니다.
- 출발일/일정표는 일정표등록에서 관리합니다.
- 공급상품은 판매사가 카테고리 연결 후 노출 체크해야 홈페이지에 노출됩니다.
