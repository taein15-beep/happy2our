# AOS 예약접수현황 기획

## 목적
예약접수현황은 여행사 직원이 예약접수 현황을 빠르게 확인하고 처리하는 운영 대시보드형 리스트 화면입니다.

## 핵심 원칙
- 예약상태와 결제상태를 분리합니다.
- 상품명과 출발일을 한눈에 확인합니다.
- 판매사 예약인지 자사 예약인지 표시합니다.
- 총금액, 결제금액, 미수금을 표시합니다.
- 알림톡 발송과 예약상태 변경을 빠르게 처리합니다.

## 상단 요약 카드
- 오늘 신규예약
- 결제대기
- 취소/환불요청
- 예약확정
- 오늘 출발
- 이번주 출발

## 검색/필터
- 기간기준: 예약접수일, 출발일, 결제일, 취소요청일
- 시작일/종료일
- 예약상태
- 결제상태
- 상품구분
- 상품명
- 출발일
- 판매채널
- 판매사
- 담당자
- 알림상태
- 미수금 여부
- 메모 여부
- 통합검색: 예약번호, 예약자명, 휴대폰번호, 상품명

## 리스트 컬럼
- 체크박스
- 예약번호 / 접수일
- 상품명 / 출발일
- 예약자 / 연락처
- 인원
- 금액: 총금액 / 결제금액 / 미수금
- 예약상태
- 결제상태
- 판매채널 / 판매사
- 담당자
- 알림 / 메모
- 관리 버튼

## 예약상태
예약접수, 예약확정, 대기예약, 예약보류, 취소요청, 취소완료, 출발완료

## 결제상태
미결제, 예약금완료, 잔금완료, 전액결제완료, 환불요청, 환불완료

## 출발자명단 정책
- 우측 상단 버튼: 검색결과 출발자명단
- 일괄처리 영역 버튼: 선택예약 출발자명단
