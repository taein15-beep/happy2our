# AOS 일정표등록 기획

## 목적
상품등록 후 출발 일정의 상세 일정을 입력하는 화면입니다.

## 기본 구조
상품등록 시 선택한 여행기간만큼 일차 탭을 자동 생성합니다.

예:
- 당일: 1일차
- 1박2일: 1일차 / 2일차
- 2박3일: 1일차 / 2일차 / 3일차

## 일정 행 구성
- 순서 변경 핸들
- 시간
- 일정유형
- 일정표 내용
- 모바일 설정
- 삭제

## 일정유형
집결, 이동, 관광, 식사, 숙박, 자유시간, 쇼핑, 선택관광, 해산, 기타

## 교통편 추가
교통수단, 교통편명, 편명, 출발지, 출발시간, 도착지, 도착시간, 참고사항을 입력합니다.

## 모바일 일정표
모바일용 일정표는 필요하지만 항상 큰 입력칸으로 노출하지 않습니다.
기본은 PC 일정표 자동반영이며, 필요 시 별도입력 영역을 펼칩니다.
