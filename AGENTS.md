# AGENTS.md

## 프로젝트
이 저장소는 AviaNext AOS 여행/항공 통합 운영 플랫폼 개발용 저장소입니다.

## 작업 규칙
- 사용자가 요청하지 않은 파일은 수정하지 않는다.
- 기존 화면 구조와 class 이름은 최대한 유지한다.
- 수정 전에는 어떤 파일을 고칠지 먼저 설명한다.
- 한 번에 여러 화면을 수정하지 않는다.
- 관리자 페이지 작업은 prototype/admin 폴더를 기준으로 한다.
- 실제 API 키, 비밀번호, 개인정보는 코드에 넣지 않는다.
- 수정 후에는 변경된 파일과 변경 내용을 요약한다.

## 중요 메뉴
- 상품관리
- 예약관리
- 정산관리
- 실적관리
- 메시지센터
- 회원관리
- 디자인설정
- 기본설정

## AOS 기획 기준
- 판매점 수수료는 정액 또는 퍼센트 중 하나만 사용한다.
- 예약금은 1인 기준으로 설정한다.
- 카드결제는 상품별 기본 ON 상태를 기준으로 한다.
- 템플릿을 불러오면 기존 내용은 덮어쓴다.
- 상세 탭은 상품별로 자유롭게 추가/삭제 가능하다.
